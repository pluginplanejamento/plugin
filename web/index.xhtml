Como utilizar o template PLUGIN


No desenvolvimento do projeto template_plugin foi utilizado o padrão MVC, onde contém os pacotes Controle: classes que irão trabalhar com as entradas dos dados; Modelo: classes responsáveis pela manipulação e armazenamento dos dados; Visão: classes responsáveis pela visualização do usuário.

O padrão de telas implementado em Web Design Responsivo deve ser integrado à arquitetura MVC. Para isso, foi estabelecida a união do HTML5, JSF e JavaScript com o pacote Visão do padrão MVC. Para tornar possível a integração do padrão MVC com o JSF, é necessário realizar um mapeamento entre a camada visual, que está em um arquivo XHTML, e o código-fonte em Java. Isso é conseguido através da anotação @ManagedBean. A classe pertencente ao pacote Visão do MVC realiza chamadas aos componentes Modelo e Controle.

No projeto contém os arquivos de CSS básicos do Foundation e o criado especificamente para o padrão: template_plugin.css, onde podem ser adicionadas mais alterações no estilo do padrão.

Antes de utilizar o template PLUGIN, é necessário que o Tomcat esteja previamente configurado em seu Eclipse, saiba como configurar através dos links: http://goo.gl/RsxRnw | https://goo.gl/OGtxjQ

Passo a passo:

1- Clonar o projeto template_plugin do GitHub: https://github.com/pluginplanejamento;


2- Abrir o package;


3- Para começar a utilizá-lo, clique no projeto "template_plugin/Java Resources/src/br.edu.ifba.plugin.PROJETO";


4- Para começar a implementação, renomeie o pacote "br.edu.ifba.plugin.PROJETO" substituindo a palavra "PROJETO" pelo nome do projeto que será implementado;
