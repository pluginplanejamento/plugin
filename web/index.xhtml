Como utilizar o Template Plugin


1- Clonar o projeto template_plugin do GitHub: https://github.com/pluginplanejamento;


2- Abrir no Eclipse, para começar a utilizá-lo clicar no projeto "template_plugin/Java Resources/src/br.edu.ifba.plugin.PROJETO";


3- Para começar a implementação, renomeie o pacote "br.edu.ifba.plugin.PROJETO" substituindo a palavra "PROJETO" pelo nome do projeto que será implementado;


A seguir, clique para visualizar o Padrão que será utilizado na interface do Projeto PLUGIN ou visualizar um exemplo de um cenário de uso.
